# Cores do Rio - Frontend API

Este é o frontend da aplicação Cores do Rio, desenvolvido com **Next.js 16** e **React 19**, utilizando **Tailwind CSS** para estilização.

Repositório Oficial: [CoresDoRio/Cores_do_rio_API](https://github.com/CoresDoRio/Cores_do_rio_API)

## Tecnologias Utilizadas

- [Next.js](https://nextjs.org/)
- [React](https://react.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [TypeScript](https://www.typescriptlang.org/)

## Como rodar o projeto localmente

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Abra [http://localhost:3000](http://localhost:3000) no seu navegador.
